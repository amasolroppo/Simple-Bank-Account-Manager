## Administrador de Recetas

Proyecto desarrollado como solución personal a un reto propuesto en un curso de Python.

Este proyecto es un administrador de recetas en línea de comandos desarrollado en Python, que permite organizar recetas por categorías, leerlas, crearlas y eliminarlas utilizando el sistema de archivos.

El objetivo del proyecto es practicar programación estructurada en Python, manejo de archivos y diseño de menús interactivos en consola.

### ⚙️ Funcionalidades
- Mostrar el número total de recetas disponibles
- Navegar entre categorías de recetas
- Leer recetas almacenadas en archivos .txt
- Crear nuevas recetas desde la consola
- Crear nuevas categorías
- Eliminar recetas existentes
- Interfaz interactiva mediante menú en terminal

### Tecnologías utilizadas
- Python 3
- Módulos estándar (os, pathlib, etc.)
- Programación modular con funciones
- Manejo de archivos y directorios
